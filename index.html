Daftar Layanan Azure yang Digunakan:

  • Azure Static Web Apps: Digunakan untuk hosting frontend statis (HTML/CSS/JS) dengan integrasi GitHub Actions.
  • Azure App Service: Digunakan untuk menjalankan aplikasi web dinamis berbasis Node.js dan Express.
  • Azure Container Apps: Layanan serverless untuk menjalankan aplikasi berbasis container (Docker image).
  • Azure Functions: Layanan Function-as-a-Service (FaaS) untuk menjalankan kode berbasis trigger (seperti API Estimasi Biaya).
